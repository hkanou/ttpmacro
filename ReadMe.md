# ttpmacro

  - [ttpmacro_mod.exe](https://github.com/hkanou/ttpmacro/tree/main/Release) -- listbox修正版  
    ttpmacro(teraterm-5.2)のlistboxに下記の修正を行ったものです。  
    - ダブルクリックによる項目選択に対応  
    - 等幅フォント使用時の表示崩れを修正  
    - ダイアログのサイズを拡大  
    - 最小化/最大化ボタンを追加  
    - アイコンを表示

    動作環境 : [Tera Term 5](https://teratermproject.github.io/)  
    ![Listbox Image](image/listbox.png)


## ビルド環境

  Visual Studio Express 2017
